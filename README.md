# :computer: How to run

```
# Clone do repositório
Faça o clone da aplicação
$ git clone https://github.com/JuanCarllos13/Todo-Ignite
```

## Rodando o projeto mobile

```
# Para rodar aplicação mobile, será necessário o EXPO

# Navegue até a pasta
$ cd TodoList

# Instale as dependências
$ npm i

# Execute o comando
$ npx expo start
```
